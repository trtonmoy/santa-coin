Getting Started

Add ECHO AI bot to your group and provide it with messaging rights. This step is crucial for the bot to operate effectively within your group.

Launching a Referral Mission

Initiate a new referral mission by sending the command "/echo_start_referral starting referral mission." This command comes with several sub-settings

Generating a Unique Referral Link

Members can create their unique referral links by using the command "/echo_ref_link." This link will be specific to each user and the active mission.

Joining Through the Referral Link

Prospective members can join your group using an existing member's referral link, which registers pending points for the referrer.

Mission Conclusion

When a mission ends, all pending and confirmed points are recalculated. The bot verifies the users' ongoing membership in the group and disqualifies any suspicious activities.
